LSculpt Help

Overall Use

To use LSculpt, you first need some kind of 3D mesh. LSculpt works with OBJ, STL and PLY.

  • Use 'File -> Import 3D File... to import your mesh.
  • Make any number of changes using the various controls in the LSculpt window, then click 'Update Preview' to see the results of your changes.
  • The model preview window is fully interactive. Click & drag with your mouse to rotate the model. Use the mouse wheel or right click and drag to zoom in & out.
  • When you are happy with your model, click 'File -> Save LDraw file As' to create an LDraw file of your finished LEGO model.

Individual Controls

Click on a control to see what it does.

Mesh Size & Position

  • Units

    This options selects the units of measurement for the input mesh. This also sets the units for the 'Distance to offset origin' and for the 'Model Scale'. Options are 'Stud', the width of a 1x1 Lego brick (approximately 8mm); 'LDraw Unit', the scale used in LDraw files (approximately 0.4mm); or a common metric or imperial measure of distance.

    For example, setting 'Units' to 'Stud' and 'Max Size' to 32 will give you a final model which measures 32 stud in its longest dimension.

  • Distance to offset origin

    These controls allow you to move the center of the imported mesh. This is useful if you adjust how the grid (and the plates in the model) lines up with some feature in the mesh.

    The offset is relative to the original mesh before any scaling.

  • Re-center Mesh at Origin

    This option will move the imported mesh to the center of LSculpt's world.

  • Up Direction & Rotation

    These control which direction the imported mesh is facing.

    • Use 'Y' or 'Z' to tell LSculpt which direction in the imported mesh is Up. If your model appears to be lying on its side, try changing the 'Up' direction.
    • Use 'Rotate Mesh' to rotate the mesh around the chosen Up direction. This changes how LSculpt lines up the layers of LEGO plates with your mesh.

  • Scaling

    These control how many studs LSculpt will use to build the LEGO model.

    • Max Size

      Use this if you want to force the overall model to be under a specific size. The Max Size is measured in the units set in the 'Units' dropdown box.

    • Scale Factor

      This is handy if you know the size of the imported mesh, and you just want to shrink or enlargen the mesh directly.

LDraw Output Options

  • LEGO Part

    This controls what LEGO part LSculpt will build your model out of.

    Options are:

    • 1x1 plate
    • 1x1 tile
    • simple 1x1 plate
    • simple 1x1 tile

    The first two options are high quality, accurate versions of their respective parts. The last two are greatly simplified versions of their parts: use either of these if LSculpt is running slow for you. You can set this to 'simple 1x1 tile', tweak your model until you are satisfied, then set it to 1x1 plate and export to LDraw for a nice finished product.

  • Part Color

    This controls how LSculpt chooses colors for your LEGO model. Note that it's not yet possible to choose specific colors directly; this control is useful as a building aid, to help you see how your model is built.

    Options are:

    • All the same Color

      Color everything the default LDraw color. In LSculpt, this will show up as light gray.

    • Color by Layer

      Change the color of each successive stack of plates. This is useful to highlight the differences between otherwise similar stacks of plates.

    • Color by Orientation

      Color all plates that are facing the same direction in the same color. This is useful to get an overall sense of the different big chunks your model is made up of.

  • Min Base Studs Up

    This tells LSculpt to make sure that a specific number of layers on the bottom of your model are facing up rather than down.

  • Empty Shell

    This tells LSculpt to only include the topmost plate in each stack of plates in your model. This is useful when you have a mesh with disconnected or overlapping surfaces that otherwise result in lots of artifacts. This can also help increase LSculpt performance, since it doesn't have to render all the plates buried inside your model.

  • Studs Up only

    LSculpt will create a model with all studs facing up.

  • Highest Detail

    Allow LSculpt to change plate orientation as much as possible, to get the best possible fit for your mesh. This might result in a model with pieces that are impossible to connect together.

  • Easiest to Build

    LSculpt will optimize the orientation of plates so that neighboring plates are more likely to face the same direction. Hopefully, this will make your model easier to build.

  • Show Plates, Show Mesh and Show Grid

    These control what LSculpt will include when it is run.

    • Plates are the final LEGO model, built with the chosen LEGO part.
    • Mesh is your original, imported 3D mesh. Turn this and plates on to see how close LSculpt matched the model to your mesh.
    • Grid is the 3D cube grid that LSculpt created internally to 'slice up' your model. You can turn on the grid and the mesh only to line up the grid with specific features on the mesh using the rotation, scale, and offset options.

  • Update Preview

    Click this whenever you want to update the model preview in the right pane, usually after making some changes to the controls on this panel.

  • Preview window

    Preview of the LEGO model. Use the mouse to move the image around:

    • Left-click and drag to spin the model
    • Middle-click (CTRL left-click) and drag to move the model (pan)
    • Right-click and drag to zoom in and out
    • SHIFT left-click and drag to change the lighting

  • Status Bar

    Warning and error messages will appear in the status bar as needed.

  • Resize Handle

    Click and drag the handle to resize the preview window.
